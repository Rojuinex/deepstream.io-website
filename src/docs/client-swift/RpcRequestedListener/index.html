Methods

onRPCRequested:data:response: required method

This listener will be invoked whenever the client recieves an rpc request from the server, and will be able to respond via RpcResponse .send(Object) or RpcResponse .reject()

Parameters:
rpcName - The name of the rpc being requested
data - The data the request was made with
response - The RpcResponse to respond to the request with
