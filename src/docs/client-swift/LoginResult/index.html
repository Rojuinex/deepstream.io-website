Methods

getData

Return the data associated with login. If login was successful, this would be the user associated data. Otherwise data explaining the reason why it wasn’t.

Returns:
A JsonElement containing the data recieved from the server during login

getErrorEvent

The error message the server sent to explain why the client couldn’t log in.

Returns:
an error event

loggedIn

Whether or not the login occurred successfully

Returns:
true if login was successful, false otherwise

init:errorEvent:data:

Called when DeepstreamClient .login(JsonElement ) is unsuccessful

Parameters:
loggedIn - false
errorEvent - error event
data - Contains data associated to the failed login, such as the reason

init:userData:

Called when DeepstreamClient .login(JsonElement ) is successful

Parameters:
loggedIn - true
userData - Optional data that is specific to the user and returned on succesfuly authentication
