Methods

onRecordChanged:data: required method

Called when the listener is added via Record .subscribe(RecordChangedCallback,boolean)
Will contain the entire record data, regardless of whether triggered by a Patch or Update

Parameters:
recordName - The name of the record change
data - The entire data as a JsonElement, can be retrieved via JsonElement .getAsJsonObject ()
