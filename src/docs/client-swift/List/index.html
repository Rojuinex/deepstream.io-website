Discard the List. This should be called whenever you are done with the List retrieved by RecordHandler .getList(String). This does not guarantee that your subscriptions have been unsubscribed, so make sure to do that first!
If all usages of the same List have been discarded, the List will no longer be updated from the server and any further usages will require the List to be retrieved again via RecordHandler .getList(String)
