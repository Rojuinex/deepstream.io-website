Methods

onSubscriptionForPatternAdded: required method

Called whenever a subscription has been found a pattern you previously added.
This method must return a true if it is willing to provide the subscription, and false otherwise. This also has to be done in a timely fashion since the server will assume the provider is unresponsive if it takes too long.

Parameters:
subscription - The name of the subscription that can be provided
Returns:
true if the server responds is willing to accept the request, false to reject

onSubscriptionForPatternRemoved: required method

If a provider has accepted a request, they will then be notified when the subscription is no longer needed so that they can stop providing

Parameters:
subscription - The name of the subscription to stop providing
