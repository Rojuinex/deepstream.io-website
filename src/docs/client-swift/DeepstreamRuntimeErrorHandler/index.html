Methods

onException:event:errorMessage: required method

Triggered whenever a runtime error occurs ( mostly async such as TimeOuts or MergeConflicts ). Recieves a topic to indicate if it was e.g. RPC, event and a english error message to simplify debugging purposes.

Parameters:
topic - The Topic the error occured on
event - The Error Event
errorMessage - The error message
