Methods

onRecordPathChanged:path:data: required method

Called when the listener is added via Record .subscribe(String,RecordPathChangedCallback,boolean)
Will contain the data under the path, regardless of whether triggered by a Patch or Update

Parameters:
recordName - The name of the record change
path - The path subscribed to
data - The data under the path as an Object
